Quickly add the canonical meta tag and customize the url.

== Description ==

Quickly enable, customize, or disable the canonical meta tag on your pages. Yoast SEO, Rank Math, and All in One SEO plugins are supported.

== Additional Info ==

This plugin allows you to quickly enable, customize, or disable the `rel="canonical"` meta tag in your posts and pages.

A canonical URL is the preferred version of a webpage that search engines like Google should prioritize over other duplicate or similar URLs. It helps prevent duplicate content in searches and ensures that search engines index and rank the most important page.

=== Features ===
- Enable or Disable if page includes the `rel="canonical"` meta tag
- Set a custom Canonical URL
- Configure the Canonical URL for the _blog home page_

=== Supported SEO Plugins ===
The following plugins work with this plugin:
- Yoast SEO
- Rank Math SEO
- All in one SEO SEO
- Slim SEO

== Installation ==

= Automatic Plugin Installation =
Install using the [built-in plugin installer](https://codex.wordpress.org/Administration_Screens#Add_New_Plugins):

1. Go to Plugins > Add New. 
2. Type in "Canonical Pages" in *Search* Plugins box. 
3. Find "Canonical Pages" in the search results.
4. Click *Install Now* to install the WordPress Plugin.
5. Click Activate to activate it.

= Manual Plugin Installation =

1. Download this plugin to your computer.
2. Extract the plugin zip file (or compressed folder).
3. With your FTP program, upload the plugin folder to the wp-content/plugins folder in your WordPress directory online.
4. Go to Plugins screen and find "Canonical Pages" plugin in the list.
5. Click Activate to activate it.

== Frequently Asked Questions ==

= Is this compatible with the Yoast SEO plugin? =
Yes, you can disable or specify a custom Canonical URL

= Is this compatible with the Rank Math SEO plugin? =
Yes, you can disable or specify a custom Canonical URL

= Is this compatible with the All in one SEO plugin? =
Yes

= Is this compatible with the Slim SEO plugin? =
Yes

= Is this plugin necessary if I have an SEO plugin? =
It may not be necessary if you are accustomed to the functionality built into your SEO plugin.

= Which SEO plugins are not supported at this time? =
Currently SEOPres, SEO Simple Pack, SmartCrawl WordPress SEO, and Squirrly SEO are not supported.

= If I set a custom Canonical URL with this plugin, does it override the SEO plugin Canonical URL? =
Yes, for the SEO plugins supported. Please see the list of supported SEO plugins.
